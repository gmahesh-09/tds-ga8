Example: Earnings Decomposition in Python


import pandas as pd

df = pd.read_csv("q4_earnings.csv")

df["core_earnings"] = (
    df["net_interest_income"]
    + df["fee_income"]
    - df["operating_expenses"]
)

df["one_offs"] = df["reported_profit"] - df["core_earnings"]

print(
    df[["quarter", "reported_profit", "core_earnings", "one_offs"]]
    .round(2)
)
Explain how a simple decomposition like this helps distinguish sustainable core earnings from one-off items, which is critical for stakeholders evaluating quality of earnings.

Financial Metrics & Valuation Formulas

Net Present Value (NPV) of future cash flows:

\( \displaystyle \text{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} \)

Return on Equity (ROE):

\( \displaystyle \text{ROE} = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} \)

Interactive Scenario Analysis

Rate shock: +100 bps parallel shift

Impact on Net Interest Income (NII)


// Simplified NII shock sensitivity (pseudo-code)
const baseNII   = 10_000_000;
const beta      = 0.35; // portfolio rate sensitivity
const rateShock = 0.01; // 100 bps

const shockedNII = baseNII * (1 + beta * rateShock);
console.log(`Shocked NII: ${shockedNII.toFixed(0)}`);
Emphasize that the real model is more complex, but this illustrates how we can codify assumptions and visualize impacts live during the presentation.
